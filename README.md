## Project Description

This project contains automated test cases for evaluating the Chat Sinhala transliteration function in the Pixelssuite Chat Translator web application.

The purpose of this assignment is to identify cases where the system fails to correctly convert chat-style Singlish input into the expected Sinhala output.

This project focuses only on the Chat Sinhala transliteration function. It does not test backend APIs, performance, scalability, security, or the Standard Sinhala transliteration function.
